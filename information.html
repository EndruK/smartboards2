About NetBoards

The NetBoards project aims to provide interactive digital noticeboards that augment the role of traditional corkboards or whiteboards.

Configuring your board

Each NetBoard can be personalised to reflect the occupants of the room. The following can be set for each individual:

  • Name
  • Description
  • Profile picture
  • Background webpage
  • Status messages
